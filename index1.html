Lists

This allows web developers to create ordened lists into their webpage, we could use it to create tables of contents, directions, etc. We have some tags to identify between an unordered list <ul> and an ordered list <ol>; also we'll need another tag in order to list <li> the topics. Example: 

An unordered list:

  • Topic
    • 

  • Topic
    • 

  • Topic
    • 

  • Topic

This is the syntax for the unordered list:

<ul>
  <li>Topic</li>
  <li>Topic</li>
  <li>Topic</li>
  <li>Topic</li>
</ul>

An ordered list:

  1. First Topic
    2. 

  2. Second Topic
    3. 

  3. Third Topic
    4. 

  4. Fourth Topic

This is the syntax for the ordered list:

<ol>
  <li>First Topic</li>
  <li>Second Topic</li>
  <li>Third Topic</li>
  <li>Fourth Topic</li>
</ol>
